### 🚀 Featured Projects

These are some of the projects I've worked on, contributing to various stages of the product development cycle, from conception and UX to coding.

*   **[WavePong](https://github.com/rinaldosbj/WavePong)**:
An immersive, retro-wave sensory game designed for accessibility. Players are guided by spatial audio and haptic feedback, making the game highly inclusive for people with visual or hearing impairments. As difficulty scales, visual elements fade, forcing total reliance on sound and touch. Developed during Apple Developer Academy.
    *   **Built with:** Swift, SwiftUI, SpriteKit
    *   **My Contribution:** Conception and development, balanced gameplay programming with accessibility UX
 
* **[Silas Pilgrim vs The CIn](https://github.com/Jorge-Guilherme/projeto-ip):**
A fun, retro 2D game inspired by university life. The main character, Silas, must navigate through the Information Center (CIn/UFPE), facing classic academic obstacles like strikes and impossibly hard problem sets, while collecting friends to advance to the next semester. Developed during Introduction to Programming course.
    * **Built with:** Python, Pygame
    * **My Contribution:** Conception and development, balanced programming with gameplay storytelling

* **[Bora Impactar](https://github.com/luizacavalcantee/Projeto-DS):**
An official platform created for Recife City Hall that connects volunteers, companies, and local NGOs to foster social impact. Created during Software Development course.
    * **Built with:** TypeScript, Next.js
    * **My Contribution:** Conception and front-end development, aligning product requirements with local government and NGO needs

* **[TaskCIn](https://github.com/analuisafeitosa/TaskCIn):**
An academic task management system that utilizes the Eisenhower Matrix to automatically classify activities based on their urgency and importance, helping students optimize their study routines. Developed during Data Structures and Object-Oriented Programming course.
    * **Built with:** C++
    * **My Contribution:** Conception and development, structured the core logic, and data handling

*   **[Ecopiticos](https://github.com/vitoriapinheiro/Ecopiticos):**
    A collaborative application built to promote and map environmental initiatives and public eco-policies within the city of Recife. Developed during Apple Developer Academy.
    *   **Built with:** Swift, UIKit
    *   **My Contribution:** Conception and development, balanced programming with UX

* **[Multi-user chat with WebSockets](https://github.com/SofiRibeiro/chat_multiusuario_com_websocket):**
A real-time, client-server chat system that allows multiple users to communicate simultaneously in a single room, featuring an automated historical message sync upon joining. Developed during Introduction to Distributed Systems and Computer Networks course.
    * **Built with:** Python, HTML
    * **My Contribution:** Developer, implemented the real-time communication architecture and connection handling
